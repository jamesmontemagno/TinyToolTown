This is a CLI tool to install reusable AI coding agent packages (agents, skills, prompts) into your repositories for consistent, enhanced AI-assisted development.

I built it because I was working with different AI coding assistants, and wanted a simple way to transfer agent files (skills, prompts, instructions) etc across different coding agents.
