FrameFit is a beautiful macOS app that lets you instantly resize any window to common device sizes or your own custom dimensions. With features like auto-centering, custom and shareable presets, and a modern glass-morphism UI, it streamlines window management for developers and power users. Built with Tauri for native performance, FrameFit is fast, lightweight, and easy to use.
