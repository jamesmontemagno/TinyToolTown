GitHub Action Dashboard is a lightweight Blazor WASM app that lets you monitor and manage GitHub Actions across multiple repos in one place. See the latest workflow status, browse recent run history, and manually trigger runs—while keeping API usage low with smart caching and manual refresh controls.
