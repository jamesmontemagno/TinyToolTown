See your iOS screen on your Windows PC with a live preview by simply plugging in a USB cable. Capture screenshots using the UI button, the tray menu, or a single command from any terminal. An AI agent can also observe your screen by repeatedly calling `--capture-frame` while it runs, enabling continuous analysis rather than just a single snapshot.
