## Ping-Legacy

**Ping-Legacy** is a modern Windows network diagnostic application that delivers a legacy-style experience for testing network connectivity and performance. It provides real-time ICMP ping monitoring with latency statistics and packet loss, supports multiple hosts simultaneously, and includes additional tools such as traceroute, DNS lookup, port and network scanning, and internet speed testing for comprehensive connection diagnostics.
