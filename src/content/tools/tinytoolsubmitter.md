TinyToolSubmitter is a standalone CLI that streamlines submitting your tool repositories to Tiny Tool Town. It supports direct execution or global installation, integrates with GitHub Copilot, and offers flexible options for automation and customization. The project includes a robust release workflow for easy publishing to NuGet and GitHub. Designed for .NET developers, it makes tool submission fast and hassle-free.
