One-command pipeline: PDF → extract text → translate → TTS → static website with audio player. Built because reading academic PDFs on the go is painful — now you can listen instead. Supports multiple languages and TTS providers. Perfect for researchers, students, and anyone who'd rather listen than squint at a screen.
