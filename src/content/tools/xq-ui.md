# XQ-UI - Interactive TUI Playground for jq/yq

## Short Description
Interactive terminal playground for testing `jq` and `yq` queries with real-time execution, AI-powered query generation, and seamless file operations. **Work-in-progress seeking contributors!**

## Full Description

**XQ-UI** is a powerful Terminal User Interface (TUI) application that transforms how developers work with JSON and YAML data. Built with Go and the Charm TUI libraries, it provides an intuitive three-panel interface for real-time query testing and data manipulation.
